OBS: TODOS OS COMANDOS NÃO TEM "ASPAS"



1º - Abra a pasta 'Site' no VS Code

2º - Agora no VS Code, aperte "Ctrl + J" para abrir um TERMINAL

3º - Escreva o comando "python -m venv venv" e aperte ENTER para criar seu próprio ambiente virtual

4º - Escreva o comando "venv\scripts\activate" para ativar a execução de scripts dentro do seu ambiente virtual.

Obs: Caso ocorra um erro nessa etapa: 
	1º - Você deve clicar no botão INICIAR do Windows com o botão direito do mouse.
	2º - Clicar em 'Windows PowerShell (Admin)'
	3º - Escreva o comando "Set-ExecutionPolicy -ExecutionPolicy RemoteSigned"
	4º - Escreva "A" e aperte ENTER
	5º - Feche o Windows PowerShell
	6º - No terminal do VS Code, escreva o comando "venv\scripts\activate" novamente.

5º - Escreva o comando "flask run"

Obs: Caso ocorra um erro nessa etapa, escreva os seguintes comandos no TERMINAL: 
	1º - pip install flask
	2º - pip install flask_mail
	3º - pip install python-dotenv

6º - Caso tenha ocorrido tudo bem, escreva novamente o comando 'flask run'

7º - Clique no link do servidor local (localHost) parecido com "127.0.0.1:5000"
